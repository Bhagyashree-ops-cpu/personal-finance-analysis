# Personal Finance Data Analysis 💰

## Overview
Analysed 3,000+ personal transactions across 4 datasets 
to uncover spending patterns and financial insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Findings
- Total spent across 21 months: $63,042
- Mortgage & Rent is the #1 expense category
- Peak spending month: June 2019
- Checking account handles 92% of all spending
- Combined and cleaned 4 different datasets

## Project Structure
- `finance_analysis.ipynb` — Complete analysis notebook
- `finance_analysis.html` — Shareable version
- `finance_dashboard.png` — Individual dataset dashboard
- `combined_finance_dashboard.png` — Combined dataset dashboard
- PNG files — Individual charts
